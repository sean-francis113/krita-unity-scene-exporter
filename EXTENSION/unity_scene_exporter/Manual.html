Unity Scene Exporter Extension

This extension was created with intended use with a Unity Importer to easily and efficently export out a full 2D Scene from Krita and into Unity.

Summary of Extension

  1. This script accesses the active document in Krita and makes sure that the document has layers in it.
  2. Once it sees that there is an active document and that it has layers, it will duplicate the document, rename it and save it to a specified location (set by the filepath, useFolders and documentFolder variables).
  3. Once it has successfully saved the document, it will then open and set active the cloned document.
  4. From there, it takes all of the top layers (the layers not inside of groups) and removes them completely.
  5. Then it goes through the rest of the layers (checking to see if there are any more layers), and merges the children of each group.
  6. After merging, it will export/save the merged layer to a specified location (set by the filepath, useFolders and exportFolder variables) with the specified extension (set by the fileExtension variable).
  7. After exporting the layer, it will then add information about the layer to an XML String.
  8. Once all of the layers have been exported properly, it will save the XML String into a XML File and the Log String into a Log File (its extension set by the logExtension variable).

Usage

The Docker has input fields where you can set various settings such as the file path to save to, file extensions, if you want to use sub folders, and more. Once those are set, hit the "Export Scene" button and let the script export everything for you!
  • Unity Scene Name: Enter the name of the Unity Scene here. The importer will name the created scene based on this name.
  • Filepath: The base filepath for exported documents.
  • File Extensions: The extensions of the files that will be exported out (images, XML and Log)
  • Use Sub-Folders: Check this if you wish to use Sub-Folders for your exports.
  • Use Batch Mode: Check this if you wish to use Batch Mode (Checked: Export Dialogues will not show; Unchecked: Export Dialogues will show)
  • Export/Document/Log/XML Folder(s): Specify the names of the sub-folders you would like to save exported files into.
  • Keywords: The list of keywords you wish the exporter to find to export out. MUST have a comma and space between them, or else the exporter will not find them.
